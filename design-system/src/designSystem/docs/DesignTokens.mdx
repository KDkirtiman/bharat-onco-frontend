# Design tokens

All components reference semantic variables from `tokens/globals.css`. **Do not hardcode brand colors** in component CSS — override tokens or use `ThemeProvider` instead.

## Themes

Use the **Theme** toolbar in Storybook (**Default** / **Contrast**) or wrap your app:

```tsx
<ThemeProvider theme="contrast">…</ThemeProvider>
```

Contrast tokens live in `tokens/themes/contrast.css` and apply via `data-ds-theme="contrast"`.
